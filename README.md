# TensorFlow-Iris-Classification

A 5-layer DNN with 4 input nodes, 10/20/10 hidden nodes, and 3 output nodes. 
Classifies an iris into:

3 possible categories
1. Iris Setosa
2. Iris Versicolor
3. Iris Virginica

Based on 4 measured features 
1. Sepal length
2. Sepal width
3. Petal length
4. Petal width
